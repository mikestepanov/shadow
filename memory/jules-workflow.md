# Jules Workflow Notes

## UI Tips

- **Always refresh the page after creating a new scheduled task** — the UI can get into stale states otherwise (e.g., submit button stuck disabled, "cancelled" state persisting)

## Known Issues (2026-02-14)

- Scheduled task modal sometimes defaults to "cancelled" status with submit button disabled
- Workaround: refresh page, try again, or use a fresh browser session
