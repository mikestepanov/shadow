## Chrome OOM Crash (19:19 CST)
- Chrome processes ballooned to ~1.4TB virtual memory each
- Kernel OOM killer terminated Chrome (PID 504821)
- **Root cause:** No swap configured on NixOS → no buffer before OOM
- System has 30GB RAM, usually fine, but no swap = instant OOM when full

## NixOS Memory Management Config Added
Added to `~/Desktop/nixos-config/configuration.nix`:
- 8GB swap file (`/swapfile`)
- earlyoom service (kills hogs at <5% RAM free, notifications enabled)
- Kernel tuning: `vm.min_free_kbytes=524288` (512MB reserved), `vm.swappiness=10`
- Prevents complete system freeze during memory pressure (macOS-like behavior)
